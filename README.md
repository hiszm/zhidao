
## 使用说明

- 环境 apache + php5以上 + mysql

- 上传wenda.sql到数据库wenda

- 后台地址 localhost/zhidao/admin.pbp

- 后台用户 admin

- 前台地址 localhost/zhidao/index.php

- 前台用户 admin moyushi cn

- 前台后台密码都是admin
